# #50 Golden Ratio Snap

**Golden Ratio Snap** is a high-speed utility for designers that calculates golden ratio proportions (1:1.618) from any single input value instantly.

## Core Philosophy
- **1 Tool / 1 Action**: Enter one number, get both larger and smaller golden proportions immediately.
- **Visual Validation**: Features a dynamic preview box that maintains the golden ratio, providing a visual sense of the proportions.
- **Frictionless Workflow**: Designed for professionals who need quick values for UI layouts, typography, or spacing without opening heavy design software.

## Features
- **Bi-directional Calculation**: Automatically generates both the "Smaller Side" and "Larger Side" based on your input.
- **Instant Copy**: Tap any result card to copy the value directly to your clipboard with clear visual feedback.
- **PC & Mobile Optimized**: Includes autofocus for rapid keyboard entry and high-visibility UI elements for all ages (accessibility-focused).
- **One-Tap Clear**: A discreet "clr" button to reset the input instantly.
- **Privacy First**: No login, no tracking, no ads. Works entirely in the browser.

## Technical Standards
- **iOS Guard**: Includes `-webkit-appearance: none` and opacity resets to ensure consistent input visibility on Safari.
- **High Visibility**: Enhanced contrast and font sizes for secondary labels to ensure readability on mobile screens.
- **Fridge Combo Design**: Pure #fff background, #333 accents, and the mandatory 15px border-radius across all interactive elements.
